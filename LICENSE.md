## 🐛 Common Issues and Fixes

### ❌ Script executes but GUI doesn't appear
**Fix:** Press `RightShift` to toggle GUI visibility. If still hidden, disable any active ReShade or graphics overlays and re-inject.

### ❌ ESP not showing player roles
**Fix:** ESP role data loads after round assignment (~3 seconds into the match). Wait for the round to fully start before activating ESP.

### ❌ Always Murderer not working consistently
**Fix:** This module works on a probability manipulation basis — it significantly increases your odds but cannot guarantee 100% outcome every round. Update to the latest version for the best results.

### ❌ Auto Farm stops collecting after a few minutes
**Fix:** The coin spawns on some maps have been patched. Toggle Auto Farm off and on to reset the path scanner, or check for a script update.

### ❌ Aimbot snaps too aggressively (looks obvious)
**Fix:** Open the GUI settings tab and reduce the **Smoothness** value and lower the **FOV** radius until the movement looks natural.

### ❌ Script doesn't work on Android via Arceus X
**Fix:** Ensure you're using **Arceus X v2.0+**. Older versions have a known incompatibility with the loader URL format. Re-download Arceus X from the official source.

### ❌ "getgenv is nil" error on execution
**Fix:** Your executor doesn't support the required environment globals. Switch to KRNL, Synapse X, or Delta for full compatibility.
